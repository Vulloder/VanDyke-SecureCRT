# Cómo instalar la aplicación

1. Instala la aplicación utilizando el archivo .exe de la carpeta «Setup»
2. Cierra la aplicación por completo (incluso desde la bandeja del sistema y el Administrador de tareas)
3. Copia los archivos crackeados de la carpeta «Claves» a la carpeta de instalación de la aplicación. Normalmente, suele ser C:\\ProgrmFiles\\VanDyke\\SecureCRT, pero si eliges la opción de Instalación Custom, podrás ver donde has descargado la aplicación.
4. Ejecuta el archivo reg
